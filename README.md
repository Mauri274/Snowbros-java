# Snow Bros Clone (Java 2D)

Clon del clásico arcade Snow Bros desarrollado en Java. Este proyecto fue construido con un enfoque en programación orientada a objetos, implementando mecánicas de juegos clásicos.

## Tecnologías y Conceptos Aplicados
* **Lenguaje:** Java
* **Requisitos:** Java 17 o superior.
* **Arquitectura:** Game Loop personalizado para el control de FPS y actualizaciones físicas.
* **Renderizado:** Java 2D Graphics.
* **Patrones de Diseño:** Implementación de concurrencia (hilos) y patrón Visitor para el manejo de colisiones.

## Estructura del Proyecto
El código fuente se encuentra en la raíz del repositorio, dividido en paquetes lógicos separando las entidades (personajes, enemigos), la lógica del juego y el motor gráfico.
